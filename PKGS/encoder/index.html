How to Use the Encoder Script in Your Own Code

This encoder is hosted at:

https://alaricholt677.github.io/PKGS/encoder/encoder.js

Using in Your Own HTML

Add this to your HTML:

<script src="https://alaricholt677.github.io/PKGS/encoder/encoder.js"></script>

Then call the encoder anywhere in your JS:

const result = alariEncode("hello world");
console.log(result);

Using in Python

Recreate the same logic in Python like this:

from encoder import alari_encode

print( alari_encode("hello world") )

(Or copy the Python version from the GitHub repo.)

Output Format

The encoder always returns:

alaricholt677.encoded$[hex value]
