🧭 Help

How to use it

This sample is designed for your fixed Spudzy Vid JS library.

Expected library script

<script src="https://alaricholt677.github.io/PKGS/spudzy-vid/spudzy-vid-js.js?v=32"></script>

Supported global names

RealLifeVideo.generate(prompt, options)
SpudzyVid.generate(prompt, options)
SpudzyVideo.generate(prompt, options)
spudzyVid.generate(prompt, options)
spudzyVideo.generate(prompt, options)

Manual example

const data = await RealLifeVideo.generate("mc voxel forest cubes sunset", {
  width: 1280,
  height: 720,
  seconds: 6,
  fps: 30,
  bitrate: 8000000,
  appendCanvas: false,
  returnFrames: false
});

document.querySelector("video").src = data.url;
