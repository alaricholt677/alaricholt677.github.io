How to Use SPUD WAVE

  1. Make sure the script line exists near the bottom of this page: 
    <script src="https://alaricholt677.github.io/PKGS/music-spud/spudzy-music-ai.js"></script>
  2. Open the page on GitHub Pages or locally in your browser.
  3. In Studio, type a title, prompt, signature, bars, lyrics, and TTS settings.
  4. Press Play. The app calls your music script like this: 
    SpudzyMusicAI.play({
  title: "My Song",
  prompt: "happy red chiptune song",
  signature: "chiptune",
  bars: 16,
  lyrics: "Spudzy rides the wave tonight",
  tts: true,
  voiceName: "",
  ttsRate: 1.08,
  ttsPitch: 1.25,
  ttsVolume: 0.95
});
  5. Press Save to save the current song in your browser.
  6. Press Export WAVE CODE to download a PNG. The PNG contains the song data in a custom grey 2D WAVE MATRIX.
  7. To import a code, use Scanner → Upload PNG / picture and select the exported WAVE CODE PNG. The app decodes it and loads the song back.
  8. If camera scanning fails, upload the original PNG instead. Camera scans need the code to be straight, close, and not blurry.

How to Use the Script in Another Page

If you only want to use spudzy-music-ai.js without the SPUD WAVE UI, include the script and call SpudzyMusicAI.play(). 

<script src="spudzy-music-ai.js"></script>

<button onclick="playSong()">Play</button>
<button onclick="SpudzyMusicAI.stop()">Stop</button>

<script>
async function playSong() {
  await SpudzyMusicAI.play({
    signature: "spudzy",
    prompt: "original red pop song with bouncy drums",
    bars: 16,
    lyrics: `
Spudzy rides the wave tonight
Red lights flashing burning bright
Every beat begins to glow
Welcome to the Spud Wave show
    `,
    tts: true,
    ttsRate: 1.08,
    ttsPitch: 1.25,
    ttsVolume: 0.95
  });
}
</script>
