Game Info

Type: Custom voxel engine

Dimensions: Overworld + Hell

Core Systems: Crafting, survival, weather, NPCs, boss, LAN

Overview

IRL Simulator is a custom-built voxel engine with block-based terrain, dynamic sky colors, a day/night cycle, weather, crafting, beds, humans, survival meters, menus, LAN hosting, and a boss spawner system.

Voxel World

Explore generated terrain with chunks, blocks, mountains, trees, caves, and structures.

Hell Dimension

Switch to a red-sky dimension with ash, bloodrock, hellstone, fire, and spawners.

Boss System

Craft and place a boss spawner, then right-click it to summon a boss.

Survival Meters

Manage water and pee meters while exploring, sleeping, and surviving.

Startup + Window Title

Startup Cinematic

When the game starts, 3D voxel blocks fly into place and form the text ALARICHOLT677.

After that, the view transitions into a blocky mountain scene where the camera comes out of a mountain and fades into the main menu.

Special Window Title

The game window title uses the player name and a random splash saying.

Format: IRL SIMULATOR 3D - PlayerName | Random Saying

Examples include Powered by Java, Grox Mode Enabled, Hydrate or Diedrate, Reality is buffering, and Your GPU is crying.

Controls

Movement

W — Move forward

A — Move left

S — Move backward

D — Move right

Mouse drag — Look around

Number keys 1 through 9 — Select hotbar slot

Interaction

B — Break selected block

P — Place selected hotbar block

Right Click — Interact, drink, open crafting, sleep, or spawn boss

T — Open chat

ESC — Pause menu

F5 — Save current level

F6 — Toggle follower

F7 — Switch dimension

Chat System

Press T to open chat, ENTER to send, and ESC to close.

Commands

/help — Show commands
/pos — Show current position
/tp x y z — Teleport to coordinates
/give id — Give selected slot a block ID
/dim or /dimension — Switch dimension
/follower — Toggle follower
/save — Save current level
/craft — Open crafting
/break — Break selected block
/place — Place selected block
/weather clear/rain/snow/thunder/random — Change weather
Older docs may mention /hell, /overworld, /spawn, or /time commands. The current command system uses /dim and /weather commands.

Crafting System

3x3 Crafting Table

Right-click a crafting table to open a 3x3 crafting grid.

Drag hotbar items into the grid. If a valid recipe is detected, an output appears.

2x2 Pause Crafting

The pause menu includes a smaller 2x2 crafting grid for quick or special recipes.

Recipe Tester

Click slots to cycle between Air, Ash, and Torch. Try 3 Ash for a Bed or 3 Ash plus 1 Torch for a Boss Spawner.

Output: Air

Bed Recipe

Ingredients

3 or more Ash blocks.

Output

Bed, Block ID 30.

Appearance

Warm beige cube block.

Boss Spawner

Recipe

3 or more Ash blocks plus 1 or more Torch blocks creates a Boss Spawner.

Output: Boss Spawner, Block ID 15.

Functionality

Place the spawner, look at it, and right-click it to spawn the boss.

The spawner works in any dimension.

Correction: Boss Spawner is ID 15. Bed is ID 30. If older notes say spawner ID 30, that is outdated.

Sleeping System

How to Sleep

  1. Craft or find a bed.
  2. Place the bed.
  3. Right-click the bed.

Sleep Effects

  • Camera tilts downward.
  • Sleep animation starts.
  • Water meter converts into pee.
  • Water resets to 0.
  • Pee increases.
  • Time jumps toward morning.
  • Wake-up message appears.

Wake-up message: <System> You wake up feeling... relieved.

Humans + Holograms

Humans

Humans are dynamic entities that can spawn in the world, save with the level, and render as entities.

Follower

Press F6 or use /follower to spawn or remove a follower.

Holographic Human

The pause menu can summon a holographic copy of a random human NPC.

Water + Pee System

Water Meter

  • Increases when drinking or interacting.
  • Drains over time.
  • Converts into pee when sleeping.

Pee Meter

  • Increases naturally.
  • Increases when drinking.
  • Increases when sleeping.
  • At 100, pee overflow happens.
When pee reaches 100, pee resets, water drains, a pee block can spawn, and the game shows: <System> You couldn't hold it...

Saving System

Manual Save

Use F5 or /save.

Saved Data

  • Player position
  • Selected hotbar slot
  • Hotbar item IDs
  • Water meter
  • Pee meter
  • Time of day
  • Overworld block data
  • Human/entity data

Worlds are saved inside configs/worldName.

Block data is stored in blockdata/blocks.dat.

Player data is stored in playerpos/playerpos.dat.

Human/entity data is stored in humanalive/entitiesAlive.dat.

LAN Multiplayer

Hosting

Hosting starts from the pause menu. The host enters a world folder inside configs and a port number.

Example: configs/MyWorld on port 25565.

LAN World Download Flow

This is what happens when joining a hosted LAN world with auto-download enabled.

Host picks world
→
Server zips folder
→
Client downloads ZIP
→
Extracts client_world
→
Game loads world
LAN demo ready.

Synced Player State

LAN sends player ID, X, Y, Z, yaw, and pitch so remote players can appear and move.
