IRL Simulator Mod Store Packs Bundle
===================================

This archive contains a /packs folder with one .zip file for every mod listed in ui.store.

Upload structure for GitHub Pages:
  modUI/ui.store
  modUI/packs/<pack_name>.zip

Total packs included: 50

These are safe starter/template packs. They contain manifests, example JSON, README files,
and placeholder data. They do NOT execute Java code.
