About This Project

Windows 12 Concept Shell is a custom Win32 GUI shell written in C++ and compiled for WinPE. It is not a real Microsoft Windows operating system. It is a conceptual environment designed to demonstrate UI ideas, custom window management, a taskbar, a start menu, apps, and a simulated desktop experience.

The shell includes:

  • Full Win32 GUI shell (no console)
  • Login system with user profiles stored in RAM
  • Desktop with draggable windows and taskbar
  • Browser with download support
  • WINstore with base64 app installer
  • Full file explorer
  • Terminal with BIOS.OPEN command
  • Notepad with tabs and JSON persistence
  • Theme system
  • Power menu triggered by Alt+F4

Important Warnings

This ISO is not a real Windows installation.

It runs entirely inside WinPE. WinPE is a minimal Windows environment intended for deployment and recovery. Because of this, several limitations apply:

  • No permanent installation to disk
  • No real user accounts (RAM only)
  • No Windows Update
  • No full driver stack
  • No full networking stack (VMware NAT recommended)
  • No real UEFI firmware interaction beyond reboot commands

Do not use this ISO as a replacement for Windows. It is a concept shell for testing, experimentation, and UI design only.

System Requirements

  • VMware Workstation or VirtualBox
  • 1 GB RAM minimum
  • WinPE-compatible boot environment
