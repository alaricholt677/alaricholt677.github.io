Using the AlariHash‑X Encoder in Your Projects

This encoder is hosted at:

https://alaricholt677.github.io/PKGS/encoder/encoder.js

Using in Your HTML / JavaScript

Add this to your HTML:

<script src="https://alaricholt677.github.io/PKGS/encoder/encoder.js"></script>

Then call the encoder in JavaScript:

const result = alariEncode("hello world"); console.log(result);

Using in Python

Install the official PyPI package:

pip install alarihash

Then use it like this:

from alarihash.encoder import alari_encode print( alari_encode("hello world") )

Output Format

The encoder always returns:

alaricholt677.encoded$[16‑digit hex value]

Database Field Naming

Recommended field names:

  • PasswordAlariHash
  • PasswordAlaricholt677Hash
  • UserPasswordAlariHashX

Important Notes

  • This is a hash, not encryption.
  • It is irreversible — the original input cannot be recovered.
  • Output length is always fixed, no matter the input size.
